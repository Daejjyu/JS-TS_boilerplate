# JS_boilerplate

### 기본 설정

```
client/.env
PORT=9000
API_END_POINT=http://localhost:3000/api

server/.env
PORT=3000
MYSQL_PWD=******
```

### 실행

```
cd client
npm install
npm run server

cd server
npm install
npm run start
```

## FE

- vanilla JS Component, Store
- webpack, webpack-dev-server (reload)

## BE

- express
- mySQL
- nodemon (reload)
